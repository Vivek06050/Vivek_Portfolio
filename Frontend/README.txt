Portfolio Application
This is a personal portfolio website created to showcase my skills, projects, and achievements in a cohesive and structured manner.

Features
About Me: A brief introduction about who I am and my professional background.
Skills: A list of my technical skills and expertise.
Projects: A showcase of my key projects, with descriptions and links to the source code.
Contact: Easy access to my contact details and social profiles.
Technologies Used
Frontend: HTML, CSS, JavaScript, React
Styling:  CSS
